📋 Complete Request-Response Flow

1. INPUT
Farmer submits query (text/image/voice) via any channel
2. VALIDATE
API Gateway validates input format and size
3. RETRIEVE
Lambda retrieves user context from DynamoDB
4. CLASSIFY
Supervisor Agent classifies query type
5. ROUTE
Route to 1-3 specialist agents based on query
6. EXECUTE
Agents execute in parallel with data sources
7. SYNTHESIZE
Supervisor combines agent outputs
8. GUARDRAIL
Bedrock Guardrails validate response safety
9. TRANSLATE
Amazon Translate converts to user language
10. STORE
Store in DynamoDB and log to CloudWatch
11. FORMAT
Format response for delivery channel
12. DELIVER
Send response to farmer via Web/WhatsApp/Telegram/Voice

🎯 MVP vs Full System

MVP (9 weeks)
Single village, 3 agents, Web only, PM-Kisan only, 50-100 users, ~$68/month
FULL (12 months)
Multi-village, 4 agents, Multi-channel, All schemes, 1000+ users, ~$500/month
